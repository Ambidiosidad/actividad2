Justificación del Framework

¿Para qué sirve?

Angular es un framework de desarrollo para construir aplicaciones web de una sola página (SPA) y aplicaciones web dinámicas.

¿Qué aplicaciones tiene?

Se utiliza para crear aplicaciones web empresariales, aplicaciones móviles híbridas, y portales web dinámicos.

¿Sobre qué lenguajes se apoya?

Angular se basa principalmente en TypeScript, una extensión tipada de JavaScript. También utiliza HTML y CSS para la estructura y el estilo.

¿Qué prerrequisitos necesita?

Los principales prerrequisitos son Node.js, npm (Node Package Manager), y conocimientos básicos de HTML, CSS y TypeScript/JavaScript.
